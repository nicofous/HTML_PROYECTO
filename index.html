¿Como realicé esta página?

Para realizar esta pagina web hice uso de dos lenguajes aprendidos en el Bootcamp Full Stack dictado por el profesor Marcos: HTML5 Y CSS3.
HTML5 es la última versión del lenguaje de marcado HTML (HyperText Markup Language). Se lanzó en 2014 y es el estándar actual para la creación de páginas web. CSS3, en cambio, es la última versión del lenguaje de hojas de estilo CSS(Cascading Style Sheets), y se utiliza para definir el diseño de las mismas.
